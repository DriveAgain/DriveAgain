##  על המשחק

**Morning Rush** 
ההוא משחק שיקומי חווייתי שמטרתו לעזור לאנשים עם פגיעת ראש לחזור בהדרגה לנהיגה בטוחה
.
המשחק משחזר את חוויית הנהיגה בצורה פשוטה, נעימה ומותאמת אישית – תוך חיזוק תפקודים ניהוליים כמו תכנון, עכוב תגובה, תשומת לב וחלוקת קשב.

בכל שלב נלמדים עקרונות נהיגה חדשים, ומתווספים אתגרים כמו תמרורים, הולכי רגל ומכשולים.

השחקן מקבל משוב מיידי, צובר נקודות, כסף וכוכבים, ויכול לשפר ביצועים ולשדרג את רכבו לאורך הדרך.

המשחק נועד לשלב חוויה של למידה, הנאה ושיקום קוגניטיבי
.
